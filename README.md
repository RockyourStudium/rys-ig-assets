# RYS Instagram-Pipeline — Bild-Assets

Öffentliche Carousel-PNGs (1080×1350) der RYS Instagram-Pipeline.
Werden von OneUp via öffentlicher URL als Carousel geladen.

- URL-Schema (GitHub Pages): `https://rockyourstudium.github.io/rys-ig-assets/png/<post-id>_<slide>.png`
- Alternativ (jsDelivr CDN): `https://cdn.jsdelivr.net/gh/RockyourStudium/rys-ig-assets@main/png/<post-id>_<slide>.png`

Generiert aus `SOCIAL_TEMPLATES/PIPELINE` (build.mjs → export.mjs). Nicht manuell editieren.
